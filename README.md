## Hello Spring

#### 摘要

本项目为以学习Spring为目的。架构为经典的SSM，采用前后端分离开发，切面编程采用AspectJ，数据库MySQL,接口设计采用RESTful API。

#### Features

- GET form/json
- POST form.json
- 小文件上传 file/base64
- 大文件下载，支持断点续传，适用于浏览器和客户端
- 切面实现打印日志、函数执行时间统计
- 增加token验证功能，新增独立的token获取和刷新接口。token验证使用自定义切面拦截。

#### Tips

- 接口返回统一为json格式
- 接口请求采用js fetch API
